# NovaTech Web Solutions Website

## Project Description
This project is a responsive single-page professional service website developed as part of Lab 14 (Open Ended Lab).
It integrates HTML5, CSS, JavaScript, and AI-generated content to demonstrate modern web development skills.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Setup Instructions
1. Clone the repository:
   git clone https://github.com/SunihaZaidi/iict-lab-14.git
2. Open the project folder in VS Code
3. Open index.html in a browser or use Live Server extension
